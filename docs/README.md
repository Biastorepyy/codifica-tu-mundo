# Carpeta de Documentos (PDFs)

Esta carpeta está reservada para los archivos PDF de los proyectos académicos y planificaciones curriculares del 3° BTI del CRECE.

## Instrucciones para Carga de Archivos:
1. Nombra los archivos PDF en minúsculas y usando guiones bajos (ej. `aranduka_crece_proyecto.pdf`).
2. Sube los archivos a este directorio.
3. Asegúrate de actualizar los enlaces correspondientes en el array de datos de `proyectos.php`.
